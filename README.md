# Warbler

Created with Flask and Jinja Template.
A Twitter clone. Users can register, followe others, be followed, write posts, and favorite posts.

## Run on your machine

```
git clone https://github.com/brendaliu2/warbler.git
flask run -p 5001
```

## Run tests
```
FLASK_ENV=production python -m unittest <name-of-test-file>
```
