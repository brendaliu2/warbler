## Tech Stack
- flask
- postgresql
- sql alchemy
- bootstrap
- jinja
- wtforms
- bcrypt

## Run on your machine

```
git clone https://github.com/brendaliu2/warbler.git
flask run -p 5001
```

## Run tests
```
FLASK_ENV=production python -m unittest <name-of-test-file>
```
